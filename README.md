# Meal Tracker

#### 01/14/2017

#### By **Ian Macharia**



This is my meal tracking application. Users can log foods they've eaten each day, including details about the meal and its number of calories.



## Setup/Installation Requirements


  * `$ git clone
  * `$ npm install`
  * `$ bower install`
  * `$ gulp build`
  * `$ gulp serve`

no bugs.
